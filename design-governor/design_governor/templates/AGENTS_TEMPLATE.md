# AGENTS

Read `HOW_IT_WORKS.md` first.

This project uses the Website Design Governor.
Default explanation mode is plain English.
Before making design edits, turn the request into a change request file and run it through the governor.
If a locked contract outside the named request would be touched, stop and report instead of drifting the design.
When reporting back, explain what happened in simple language first, say whether the change passed or was blocked, give the next safe action, and put technical details last.
